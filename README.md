# Date & Period Utility in C++

This C++ program provides a simple **Date and Period utility** for reading dates, calculating differences, and computing period lengths in days. It supports leap years and can include or exclude the end date in calculations.

---

## Features

- **stDate structure**: Represents a date with `Year`, `Month`, and `Day`.
- **stPeriod structure**: Represents a period with a `StartDate` and `EndDate`.
- **Date Comparisons**: Check if a date is before, equal, or after another date.
- **Leap Year Check**: Determine if a year is a leap year.
- **Days in Month**: Automatically calculates the number of days in each month.
- **Period Calculations**:
  - Compute the length of a period in days.
  - Optionally include the end date.
- **Interactive Input**: Functions to read a day, month, year, full date, or a period from the user.
- **Date Increment**: Increase a date by one day, correctly handling month/year transitions.

---

## Usage

1. **Open in Visual Studio or any C++ IDE**.
2. **Create a Console Application project**.
3. **Add `main.cpp`**.
4. **Build and run the program**.
5. **Follow the prompts** to enter a period, and see the calculated period length in days.
